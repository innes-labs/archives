#### **SERVER MACHINE PREREQUISITE**

	- Note: better when supporting GPU

	- Windows Server
- DDR size: 
	- 8 GB minimum:
		- 4 GB for Briva + 
		- 4 GB for Windows Server
- Firewall input traffic rule: 
	- TCP, port 80 for http and
