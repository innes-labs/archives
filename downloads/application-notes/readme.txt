VNC demo V1.10.11 started guide

VNC Demo is a demonstration tool to capture the desktop of a PC, each 1 second, to be able to view it on a remote player.

Limitations :
- Tested on Windows XP, VISTA and Windows 7, with administrator privileges



Unzip the file, on the desktop-PC you want to capture.
Launch 'demoVnc.exe' (accept to unblock any security feature)

On an INNES playzilla G3, use the following url to see the content :
http://<IP>:5800/vnc.html, where <IP> is the IP address of the desktop PC
Example :
http://192.168.1.41:5800/vnc.html
